# Zero Point

**Zero Point** — это 2D-платформер, в котором вы управляете роботом, случайно провалившимся в недра старой фабрики. Чтобы выбраться обратно на конвейер, герою предстоит преодолеть множество препятствий и использовать уникальные особенности своего корпуса.

---

## 🎮 Геймплей

Игрок проходит три уровня фабрики, поднимаясь после каждого на ярус выше. На пути встречаются движущиеся прессы, шипы, платформы и скрытые участки, которые требуют внимательности и правильного использования способностей робота.

---

## 🧠 Основные механики

### Базовое управление
- Движение влево/вправо
- Прыжок

### Уникальные способности
- **Режим магнита** — временное прилипание к металлическим поверхностям, покрытым машинным маслом.
- **Сканирование окружения** — подсвечивает скрытые платформы на 2 секунды.

---

## ⚙️ Управление

| Действие | Клавиша |
|----------|---------|
| Движение влево / вправо | `A` / `D` |
| Прыжок | `W` |
| Режим магнита (зажать) | `Left Shift` |
| Сканирование окружения | `E` |
| Выход из игры | `Esc` |

---

## 🎨 Визуальный стиль

Приглушённые металлические тона — серый, коричневый, ржаво-красный — создают атмосферу заброшенной фабрики.

---

## 🗂️ Структура проекта

| Файл | Описание |
|------|----------|
| `Game1.cs` | Главный игровой цикл, обработка ввода, загрузка контента |
| `Player.cs` | Робот: движение, прыжки, состояния |
| `MagnetMode.cs` | Логика прилипания к металлическим поверхностям |
| `ScanSystem.cs` | Активация сканирования и таймер подсветки |
| `Obstacle.cs` | Базовый класс препятствий |
| `MovingPlatform.cs` | Движущиеся платформы |
| `PressTrap.cs` | Движущийся пресс с зоной поражения |
| `SpikeTrap.cs` | Шипы |
| `HiddenPlatform.cs` | Скрытая платформа (проявляется при сканировании) |
| `LevelManager.cs` | Загрузка уровней 1–3, переход между ярусами |
| `Camera.cs` | Камера, следящая за игроком |
